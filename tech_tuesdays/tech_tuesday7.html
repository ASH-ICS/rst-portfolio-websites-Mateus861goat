What is Blockchain? Blockchain is a ledge that works almost as a full history of transactions for Cryptocurrencies like Bitcoin. Each new transaction that happens with the Cryptocurrency is recorded in a block, where the information gets encrypted. Each new block attaches to the last one, making a chain of blocks, hence the name Blockchain. The purpose of blockchain: Blockchain allows for cryptocurrencies such as Bitcoin to not be regulated by a central authority, but instead to have the transactions in the currency verified by all the users, who can see all the transactions live at blockchain.com. Each computer that is connected to the the blockchain network receives a copy of the blockchain upon connecting to it.
